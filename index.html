Java Project 2022: Scuba Survivors

My java project consists of a scuba diver, slimes, sirens and pearls. The lone scuba diver finds themselves in the deepest depths of the ocean, holding fast to their one goal: discover and pocket all the riches of the ocean. My platformer game displays thoroughly understanding of object-orientated programming through the uses of inheritance and encapsulation. This project helped develop a better understanding of said object-orientated programming, GUIs, and Java in general.
